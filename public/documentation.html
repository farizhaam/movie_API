Request URL HTTP Method Request data format Response data format

Get data about a single movie by title /movies/[Title] GET None A JSON holding data about a single movie,
Example:

{
"Genre":
"Name": "Genre's name",
"Description": "Description of the genre.",
"Director: "
"Name": "Director's Name",
"Bio": "Director's biography",

"_id": objectId,
"Title": "Movie Name",
"Description": "Movie description",
"Actors": ["Actor 1", "Actor 2"],

Get a list of all genres /genres GET None A JSON object holding data about all the movie genres
Get data about a movie genre by name /genres/[Name] GET Specify genre name in URL. The query is case sensitive. A JSON object holding data about a single movie director,
Example:

{
"Name": "Genre's name",
"Description": "Description of the selected genre..."
}

Get a list of all movie directors /directors GET None A JSON object holding data about all the movie directors
Get data about a single movie director by name /directors/[Name] GET Specify director name in URL. The query is case sensitive. A JSON object holding data about a single movie director,
Example:

{
"_id": objectId,
"Name": "Director's Name",
"Bio": "Director's biography",
"Birthyear": "YYYY",
"Deathyear": "YYYY"
}

Get a list of all users /users GET None A JSON object holding data about all the registered users
Get data about a user by name /users/[Name] GET Specify director name in URL. The query is case sensitive. A JSON object holding data about a single user,
Example:

{
"_id": objectId,
"Name": "Firstname Lastname",
"Username": "Username",
"Password": "Password",
"Email": "example@mail.com",
"Birthday": "YYYY-mm-dd",
"FavoritesMovies": [MovieID1, MovieID2]
}

Allow new users to register /users POST A JSON object holding data about the new user to add,
structure like this:

{
"Name": "Firstname Lastname",
"Username": "Username",
"Password": "Password",
"Email": "example@mail.com",
"Birthday": "YYYY-mm-dd",
}

A JSON object holding data about the user that was added, including an ID:

{
"_id": objectId,
"Name": "Firstname Lastname",
"Username": "Username",
"Password": "Password",
"Email": "example@mail.com",
"Birthday": "YYYY-mm-dd",
}

Allow users to update user info by username /users/[Username] PUT A JSON object holding data about user info to update,
Example:

{
"Email": "new.email-address@mail.com", }

A JSON object holding data of a user info that is sucessfully updated,
Example:

{
"_id": objectId,
"Name": "Firstname Lastname",
"Username": "Username",
"Password": "Password",
"Email": "new.email-address@mail.com",
"Birthday": "YYYY-mm-dd",
}

Allow users to add a movie to favorite list /users/[Username]/movies/[MovieID] POST None A JSON object holding data of a user info that is sucessfully updated,
Example:

{
"_id": objectId,
"Name": "Firstname Lastname",
"Username": "Username",
"Password": "Password",
"Email": "new.email-address@mail.com",
"Birthday": "YYYY-mm-dd",
"FavoritesMovies": [MovieID1]
}

Allow users to remove a movie from favorite list /users/[Username]/movies/[MovieID] DELETE None A text message that indicating the movie was successfully removed from favorite list.
Allow users to de-register /users/[Username] DELETE None A text message that indicating the user was successfully de-registered.
